# FAQ

**Q : Comment créer un compte ?**
R : Cliquez sur “S’inscrire” et suivez les instructions.

**Q : Comment réinitialiser mon mot de passe ?**
R : Utilisez le lien “Mot de passe oublié” sur la page de connexion.

**Q : Mes données sont-elles sécurisées ?**
R : Oui, toutes les données sont chiffrées et protégées.

**Q : Qui contacter en cas de problème ?**
R : Contactez l’administrateur via la page de contact ou l’email indiqué dans les mentions légales.
